# Wareon Analytics

**Система аналитики нового поколения** — Telegram-бот, который не просто считает цифры, а помогает принимать бизнес-решения.

## Направления

| Модуль | Что умеет |
|---|---|
| 📊 **Бизнес-аналитика** | учёт продаж, прибыль и маржа, конверсия, средний чек, ROI/ROMI, точка безубыточности, расчёт зарплаты, воронка продаж |
| 📣 **Аналитика соцсетей** | добавьте бота в канал или группу — он собирает статистику: сообщения, посты, подписки/отписки, активные участники, прирост |
| 🛍 **Маркетплейсы** | воронка карточки (CTR → корзина → заказ), ДРР, юнит-экономика — для дизайнеров карточек и менеджеров МП |
| 📑 **Умные таблицы** | пришлите .xlsx/.csv — бот разберёт структуру, посчитает суммы/средние, найдёт топ-категории и пустые ячейки |
| 📋 **Отчёты** | сводка по продажам за период с графиком выручки по дням и рекомендацией, что делать дальше |

## Быстрый старт

```bash
# 1. Установить зависимости (Python 3.11+)
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

# 2. Настроить токен
cp .env.example .env
# впишите BOT_TOKEN из @BotFather

# 3. Запустить
python -m wareon.main
```

## Основные команды бота

```
/start                          — меню и справка по разделам
/sale 15000 8000 сайт           — записать продажу
/profit 100000 60000            — прибыль, маржа, наценка
/conversion 1000 37             — конверсия
/avg 150000 42                  — средний чек
/roi 50000 200000               — ROI
/romi 300000 100000             — ROMI
/breakeven 100000 1500 900      — точка безубыточности
/salary 50000 800000 5 10000    — зарплата: оклад + % с продаж + бонус
/funnel показы:10000 клики:800 заказы:56 — воронка продаж
/card 50000 2500 400 120        — воронка карточки маркетплейса
/unit 1500 20 80 600 50         — юнит-экономика МП
/report 30                      — отчёт с графиком за 30 дней
/channels                       — статистика подключённых каналов/групп
/stats                          — статистика (внутри группы)
/tables                         — история загруженных таблиц
```

Для аналитики соцсетей добавьте бота администратором в канал или группу — сбор статистики начнётся автоматически.

## Архитектура

```
wareon/
├── main.py           # точка входа: запуск polling
├── config.py         # настройки из .env
├── keyboards.py      # inline-меню
├── db/               # SQLite + SQLAlchemy (async): пользователи, продажи,
│                     # отслеживаемые чаты, события, загрузки таблиц
├── services/         # чистая бизнес-логика (покрыта тестами)
│   ├── analytics.py  # все бизнес-расчёты
│   ├── social.py     # агрегация статистики чатов
│   ├── tables.py     # разбор и сводка CSV/XLSX
│   └── reports.py    # отчёты и графики (matplotlib)
└── handlers/         # обработчики Telegram по разделам
```
